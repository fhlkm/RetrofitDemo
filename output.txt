 <h1 class="title">Hello, World!</h1>  
 <div class="main-content-block">
 <div id="node" style="font-family:inherit" data-format="2" class="node node-nid-8">
 <div style="float:right">

 <div class="content clear-block node-book" id="content-container"> <p>JavaScript is an interpreted language. That means: no compilation. You just write code and attach it to HTML.</p>
 <p>The simplest way is to put JavaScript directives into <code>SCRIPT</code> tag somewhere inside the page.</p>
 <p>Here is a <code>SCRIPT</code> tag containing single JavaScript statement. Please click the &#8216;Show&#8217; button to see the example.</p>
 <p><pre class="source brush:js; html-script:true;height:100;run:true" style="">
 &lt;!DOCTYPE HTML&gt;
 &lt;html&gt;
 &lt;body&gt;
   &lt;p&gt;Header...&lt;/p&gt;
   &lt;script&gt;
     alert('Hello, World!')
   &lt;/script&gt;
   &lt;p&gt;...Footer&lt;/p&gt;
 &lt;/body&gt;
 &lt;/html&gt;
 </pre></p>
 <p>Note that contents of the <code>SCRIPT</code> block is not shown, instead, it is executed.</p>
 <p>This example makes use of following elements:</p>
 <dl>
 <dt><code>&lt;script&gt; ... &lt;/script&gt;</code></dt>
 <dd><code>&lt;script&gt;</code> tag tells browser that there is executable content inside. In older HTML standard, namely HTML4, a special attribute <code>type</code> was required. Although more up-to-date HTML5 makes it optional.
 <p>Browser:</p>
 <ol>
 <li>Starts showing the page, shows <code>Header</code></li>
 <li>Switches to JavaScript mode inside this tag and executes the contents of <code>&lt;script&gt;</code></li>
 <li>Returns back to HTML-mode and continues with the page, outputs <code>Footer</code>.</li>
 </ol>
 </dd>
 <dt><code>alert(...)</code></dt>
 <dd>Outputs a message window on the screen and awaits until visitor presses OK</dd>
 </dl>
 <p>Not only reading, but actually testing and doing something on your own is very important. Below you can see a &#8220;check yourself&#8221; block. It contains a task and the solution.</p>
 <p>Fulfil the task to ensure you got eveything right. </p>
 <p><div style="font-family:inherit" data-format="2"><div class="task task-nid-20">
 <div class="task-formulation"> <p>Create a page which outputs a &#8220;I am JavaScript!&#8221;. Save the document and open in the browser. Make sure everything works.</p>
 <p>Or, in other words, create the page which works like: <a href="http://lookatcode.com/files/tutorial/browser/script/alert/index.html" class="liplay" target="_blank">tutorial/browser/script/alert/index.html</a>.</p>
 </div>
 <span class="task-solution-open">Open solution</span>
 <div class="task-solution">
 <div class="task-solution-header">Solution</div>
 <p>Open source code of the given example to see the solution or just here: <a href="/play/tutorial/browser/script/alert/index.html" class="liplay" target="_blank">tutorial/browser/script/alert/index.html</a>.</p>
 </div>
 </div></div></p>
 <p>On next pages we&#8217;ll go deeper into the examples and basic script building blocks.</p>
